# Animated 8-BIT GF Clock 👻🕒

A retro-style animated digital clock built with Pygame.

Features:
- 12-hour or 24-hour time format
- Per-digit animated transitions
- Config-driven settings (no code edits required)
- Optional fullscreen mode
- 8-bit ghost character animation
- System font rendering

---

## Requirements
- Python 3.9 or newer
- Pygame

---

## Install

```bash
pip install -r requirements.txt
